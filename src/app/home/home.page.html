Página de Apresentação

Esta é a página de apresentação da Avaliação Formadora 2. Esta página contém dois botões para as outras duas páginas exigidas para esta avaliação.

Cada botão leva a uma página que contém um propósito:
"Página API" leva a uma página cujo único propósito é mostrar o uso da API selecionada.

"Página ALUNO" leva a uma página cujo único propósito é mostrar o aluno responsável por essa avaliação.
