Pagina API

Esta página contém uma API que gera uma citação aleatória de algum personagem da série Game Of Thrones.

Site da API: https://gameofthronesquotes.xyz/

Os dados da citação são gerados em um Json e o meu componente filtra as informações: Citação e Personagem e os mostra aqui.
